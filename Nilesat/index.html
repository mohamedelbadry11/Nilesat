Key Satellite Formulas

Antenna Gain (G): G = 10·log₁₀(η·(π·D/λ)²
Free Space Loss (FSL): FSL = 20·log₁₀(4πR/λ)
EIRP: EIRP = PTX + GTX - LTX
G/T: G/T = GRX - 10·log₁₀(Tsys)
Thermal Noise (N): N = k·Tsys·B
C/I: C/I = C - I + D + Xpol - B
Orbital Period: T = 2π√(a³/μ)
Velocity: v = √(μ(2/r - 1/a))
Coverage Angle: θ = cos⁻¹(R/(R+h))
Footprint Diameter: D = 2h·tanθ
Beamwidth: θ = 70λ/D (degrees)
Path Loss: L = FSL + Atmospheric Losses
Doppler Shift: Δf = (v·f·cosθ)/c
Rain Attenuation: A = a·Rb·L·k
